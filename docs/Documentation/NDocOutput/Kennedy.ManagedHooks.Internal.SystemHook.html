Remarks

To use this class you typically create a derived type which is a particular type of hook. Then simply implement the HookCallback method. This method will be called whenever an event is caught by the base class. For example, we have created the MouseHook and KeyboardHook classes following this prescription.

Note that each type of hook class is expected to exist as a singleton.
