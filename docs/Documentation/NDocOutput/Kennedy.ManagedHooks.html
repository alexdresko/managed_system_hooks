Namespace hierarchy

Classes

Class Description
HookFactory The hook factory class is a general hook interface factory which creates class instances supporting the requested interfaces.
HookTypeNotImplementedException An exception indicating that there is already a class hooking a particular event type.
KeyboardHookEventArgs Keyboard hook event argument class.
ManagedHooksException This class represents exceptions that originate from the ManagedHooks classes.
MouseHookEventArgs Mouse hook event argument class.

Interfaces

Interface Description
IKeyboardHook This interface represents a system hook of the type KeyboardLL (low level keyboard system events).
IKeyboardHookExt This interface represents a system hook of the type KeyboardLL (low level keyboard system events).
IMouseHook This interface provides access to low level mouse hook events. For typical use the IMouseHookLite interface is a better choice in place of this interface.
IMouseHookExt Provides access to higher level mouse hook events such as the mouse move, button events, and wheel events. Use IMouseHookLite in place of this interface if you do not want to track the MouseMove event.
IMouseHookLite The IMouseHookLite interface provides access to basic mouse hook events but specifically does not monitor MouseMove events. Prefer this interface when you do not need to track mouse movement (only mouse button events and mouse wheel events).
ISystemHook The common interface shared among all system hook interfaces.
