MouseHookImpl.MouseEvent Event

This event is fired whenever a system hook event is received by the MouseHook class.

[Visual Basic]
Public Event MouseEvent As MouseEventHandler
[C#]
public event MouseEventHandler MouseEvent;

Event Data

The event handler receives an argument of type MouseHookEventArgs containing data related to this event. The following MouseHookEventArgs properties provide information specific to this event.

Property Description
Alt Indicates whether the alt key was pressed during the event.
Ctrl Indicates whether the control key was pressed during the event.
Empty A static instance that can be used when no information is available. For example, during the mouse wheel events.
Location The location of the cursor when the event occurred.
Shift Indicates whether the shift key was pressed during the event.
