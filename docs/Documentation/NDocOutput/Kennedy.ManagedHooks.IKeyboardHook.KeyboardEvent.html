IKeyboardHook.KeyboardEvent Event

Event triggered when a keyboard hook related system hook event occurs.

[Visual Basic]
Event KeyboardEvent As KeyboardEventHandler
[C#]
event KeyboardEventHandler KeyboardEvent;

Event Data

The event handler receives an argument of type KeyboardHookEventArgs containing data related to this event. The following KeyboardHookEventArgs properties provide information specific to this event.

Property Description
Alt Indicates whether the alt key was pressed during the event.
Cancel Indicates whether to cancel the key event for the foreground application.
CapsLock Indicates whether the caps lock key was pressed during the event.
Ctrl Indicates whether the control key was pressed during the event.
Key The key that has triggered the event.
Shift Indicates whether the shift key was pressed during the event.
