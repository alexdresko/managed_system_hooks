VirtualKeys Enumeration

Represents the keys from keyboard events received from Win32 events.

[Visual Basic]
Friend Enum VirtualKeys
[C#]
internal enum VirtualKeys

Remarks

It would be ideal if we could simply use the System.Windows.Forms.Keys enumeration. But there is no direct translation from the integers received from the C++ layer (Win32 key codes) and this enumeration. Thus we have included the VirtualKeys enumeration along with the ConvertKeyCode to bridge that gap.
