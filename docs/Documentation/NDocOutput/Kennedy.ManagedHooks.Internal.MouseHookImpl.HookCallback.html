The WPARAM parameter associated with the system hook event.

The LPARAM parameter associated with the system hook event.

Remarks

These three parameters vary by system hook event. For example, in the HookTypes.MouseLL event, wparam is typical WM_LBUTTONDOWN which maps to the enumeration MouseEvents.LeftButtonDown. See the Win32 API documentation for SetWindowsHookEx for details about what these parameters mean in the various hook types.
